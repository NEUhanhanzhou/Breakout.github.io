Detailed Description

This class sets up a full graphics program
Initialization: initMedia()->createLevel()->initUI()->selectLan().
Loop: Update()->Render().
Before the game start, the player has to choose english or japanese.
After that, the game enter the loop and keep updating and rendering the screen .
Whenever passing a level (all bricks all removed), the game will load level configuration files automatically and wait for 3 seconds to enter the next level.
If all levels are passed, display the win message and system will pause. If the number of lives is 0, then display the lose message and restart.
